TROUBLESHOOTING:
================================================================================

Problem: "git is not recognized"
Solution: Install Git from https://git-scm.com/downloads

Problem: "Permission denied"
Solution: Make sure you're logged into GitHub
         Run: git config --global user.name "Your Name"
         Run: git config --global user.email "your@email.com"

Problem: "Repository not found"
Solution: Make sure the repository exists on GitHub
         Create it first at: https://github.com/new

Problem: "Failed to push"
Solution: Check your internet connection
         Verify repository URL is correct
         Make sure you have push access

Problem: ".env file is being uploaded"
Solution: Make sure .env is in .gitignore
         Run: git rm --cached .env
         Run: git commit -m "Remove .env from tracking"
