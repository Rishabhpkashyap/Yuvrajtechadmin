================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Repository not found"
Solution: GitHub pe pehle repository banao manually

Problem: ".env file upload ho raha hai"
Solution: Use uncheck karo ya .gitignore check karo

Problem: "Authentication failed"
Solution: GitHub Desktop se logout karke phir login karo

Problem: "Large files"
Solution: node_modules folder delete karo, phir try karo
